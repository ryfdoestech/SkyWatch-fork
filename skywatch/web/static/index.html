USB driver helper (Zadig)
Replace your RTL-SDR's driver with WinUSB so SkyWatch can talk to it.
To find your RTL-SDR in Zadig: click Options → enable List All Devices, then uncheck Ignore Hubs or Composite Parents. The dongle usually appears as Blog V4 (or Bulk-In, Interface (Interface 0) on older units). Set the target driver to WinUSB and click Replace Driver.

WiFi capture driver (Npcap)
Required for the WiFi-monitor-mode Drone Remote ID path. Bluetooth Drone-RID works without it.
In the Npcap installer: accept the license, leave the defaults checked (especially Install Npcap in WinPcap API-compatible Mode and Support raw 802.11 traffic for wireless adapters), click Install. After it finishes, restart SkyWatch and click Re-check above.
